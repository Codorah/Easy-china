# 🇨🇳 Easy China – Site Vitrine Premium

Partenaire d'import, études et visa depuis le Togo, la Réunion et toute l'Afrique vers la Chine.

## Stack
- React 18 + Vite 5
- lucide-react (icônes)
- localStorage (persistance admin)
- Unsplash (images dynamiques)

## Démarrage local
```bash
npm install
npm run dev
```

## Déploiement
Le site se déploie automatiquement sur Vercel à chaque push sur `main`.

## Admin
URL : https://easychina-services.com/#admin  
Mot de passe : [défini en variable d'env VITE_ADMIN_HASH]
